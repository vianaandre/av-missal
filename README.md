## Como obter o código fonte

1. Criando uma pasta

   ```sh
   mkdir av-missal
   ```

2. Entrando na página

   ```sh
   cd av-missal
   ```
   
3. Iniciando o git

   ```sh
   git init
   ```

4. Fazendo o pull do Código

   ```sh
   git pull url-repositorio
   ```
